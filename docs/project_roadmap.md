# Project Roadmap

## Phase 1 — Data Foundation
- Replace sample data with real 2026 prospect, combine, mock, team, and draft-order data.
- Create source notes for every non-official field.
- Validate IDs, teams, and pick ownership.

## Phase 2 — Basketball Feature Engineering
- Convert combine metrics into role scores.
- Build team need profiles from roster construction, age curve, contracts, and recent performance.
- Normalize production by league strength and age.

## Phase 3 — Predictive Models
- Prospect Value Model
- Team Fit Model
- Mock Consensus Model
- Availability Simulation
- Trade Risk Model

## Phase 4 — Backtesting
- Use historical years to evaluate exact pick accuracy, pick-range accuracy, MAE, team match, and position match.
- Compare against baselines: consensus-only, big-board-only, team-fit-only.

## Phase 5 — Analyst Report and Portfolio Packaging
- Final board with confidence and alternatives.
- Model card with assumptions and limitations.
- Short article explaining project motivation, data, and findings.
